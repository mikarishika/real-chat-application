# 💬 Real-Time Chat Application

A modern real-time chat application built with **Node.js, React, and MongoDB**, supporting instant messaging and real-time communication using WebSockets.

---

## 🚀 Features

- ⚡ Real-time messaging with Socket.io
- 👥 Online users detection
- 💬 Private chat system
- 🔐 Secure backend with Node.js & Express
- 📦 MongoDB database integration
- 🎨 Responsive and clean UI with React
